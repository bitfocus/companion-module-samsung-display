## Samsung Display MD series

**Available commands in this module**

- Power on
- Power off
- Input
- Mute
- Volume
- Contrast
- Brightness
- Sharpness
- Saturation
- Tint
- Video wall state
- Video wall mode
- Custom command

**Available feedbacks in this module**

- Power
- Input
- Mute
- Volume
- Contrast
- Brightness
- Sharpness
- Saturation
- Tint
- Video wall state
- Video wall mode
- Video wall screen number

**Available variables in this module**

- Power
- Volume
- Mute
- Input
- Model
- Screen size
- Software
- Serial number
- Contrast
- Brightness
- Sharpness
- Saturation
- Tint
- Fan speed
- Video wall mode
- Video wall state
- Video wall monitor number

**Configuration**

You may need to turn on the following, to allow the module to turn displays back on when they've been turned off.

 - System
   - Power Control
     - Remote Configuration
